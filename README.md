# Collatz-Checker
A quick code you can run to check if numbers fit the Collatz Conjecture.

# !! IMPORTANT !!
This requires the rich library to function. You can learn how to install it [here.](https://pypi.org/project/rich/) It also requires the [pyfiglet](https://pypi.org/project/pyfiglet/) and [pynput](https://pypi.org/project/pynput/) libraries.

# Python version:
3.11.5
